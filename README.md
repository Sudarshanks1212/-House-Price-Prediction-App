# -House-Price-Prediction-App
🏠 House Price Prediction App

This project is a machine learning-based web application that predicts house prices based on property features such as size, number of rooms, and neighborhood quality. The app is built using a regression model and deployed with Streamlit.

# 🚀 Features
Predict house prices instantly
User-friendly single-page interface
Real-time input and output
Download prediction results
Clean and simple UI

# 🛠️ Technologies Used
Python
Pandas, NumPy
Scikit-learn
Streamlit

# 📂 Project Structure
├── app.py

├── model.pkl

├── scaler.pkl

├── requirements.txt

└── house_price_regression_dataset.csv
 
# 🌐 Live Demo : https://house-price-prediction-1212.streamlit.app/



# 📊 Model Details
Model: Linear Regression
Preprocessing: Standard Scaler
Evaluation: MAE, MSE, RMSE, R² Score

# 🎯 Use Case
This application helps users estimate property prices quickly based on key housing features.
